# Example Skill (template)

Esta skill é um esqueleto. Use como ponto de partida ao criar uma skill nova.

## Quando usar

Descreva aqui os gatilhos concretos: tipos de pedido, palavras-chave, contextos do projeto. O Claude usa o `description` do frontmatter pra decidir se carrega esta seção, mas vale repetir/expandir aqui o "quando" pra reforçar.

## Como proceder

1. Passo um — o que fazer primeiro.
2. Passo dois — checagens, validações, comandos.
3. Passo três — entregar o resultado.

## Regras

- Use listas curtas e diretas.
- Inclua exemplos quando o passo for ambíguo.
- Aponte pra arquivos auxiliares (`reference.md`, `scripts/`) se o conteúdo for longo.
